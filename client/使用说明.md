
## 快速启动

### 第一步：启动服务器

打开终端，进入 server 目录：

```
cd server
npm start
```

服务器启动后显示：`Server running at http://127.0.0.1:3001`

### 第二步：启动客户端（生产模式）

另开一个终端，进入 client 目录：

```
cd client
npm run preview
```

浏览器访问：`http://localhost:5173`

---

## 注册与登录

- 首次使用需要注册账号，填写用户名、密码、昵称
- 每个账号初始资金 **100 万元**
- 登录后 Token 自动保存，7 天内免重新登录

## 主要功能

### 股票交易

**交易页面**提供实时行情和交易操作：

- **市价单**：以当前价格立即成交
- **限价单**：设定委托价格，等待价格到达时成交
- **数量**：最小单位 100 股，可使用 25% / 50% / 75% / 100% 快捷按钮
- 手续费率：**0.03‰**（万三）

**T+1 规则**：当天买入的股票当天不能卖出，次日方可卖出（冻结显示为"T+1冻结 N 股"）
